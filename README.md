# NOMADE

**Julieta Carballido - N° 354016**

NOMADE es una tienda de muebles de diseño. Mantiene una estética limpia y prolija, con colores blanco y negro, inspirada en una estética editorial.

---

# Descripción de la página

## INDEX

* Pantalla de inicio con:

  * Nombre de la tienda
  * Breve slogan

* Contador con fecha límite de un descuento

* Marquee de productos destacados

* Tres productos destacados que cambian al refrescar la página

* Formulario de inscripción a la página

---

## CATÁLOGO

### Filtro de búsqueda en tiempo real

* Filtra por categoría del producto
* Filtra por nombre del producto
* Cartel de:

  * `"No hay resultados para su búsqueda"`

### Filtros de selección

* Selección de categoría de productos
* Filtro de productos `"en temporada"`

### Listado de productos

Cada producto incluye:

* Cartel de `"Últimas unidades"`
* Cartel de `"En temporada"`
* Categoría del producto
* Nombre del producto
* Imagen del producto
* Botón `"Ver más"`

---

## AMPLIACIÓN

Vista detallada del producto con:

* Nombre
* Descripción
* Stock disponible
* Botón `"Agregar al carrito"`
* Precio
* Imagen principal
* Imágenes secundarias

### Productos relacionados

* Listado de cuatro productos relacionados filtrados por categoría
